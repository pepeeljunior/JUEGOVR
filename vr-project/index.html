Experiencia Meta Quest 3

Diseñado con soporte WebXR nativo para Meta Quest 3. Abre el juego desde el navegador de tu visor y presiona "ENTER VR" para inmergirte en el mundo. Los controles de mando detectan joystick y gatillos automáticamente. En PC puedes jugar con teclado y ratón.

Manual de explorador

Controles

🚶
Caminar
WASD para mover al personaje.

VR: Joystick izquierdo para moverse.
🏃
Correr
Mantén SHIFT mientras caminas para correr.

VR: Presiona el stick izquierdo.
⬆️
Saltar
ESPACIO para saltar.

VR: Botón A del mando derecho.
💃
Bailar
Presiona B para activar la animación de baile.

VR: Botón B del mando derecho.
↩️
Girar 90°
Q girar izquierda · E girar derecha (90° exactos).

VR: Joystick derecho izq/der.
📷
Cámara
Mueve el ratón para rotar la cámara alrededor del personaje.

VR: El headset controla la vista.
